INSTRUÇÕES PARA O PROJETO

Após baixar o repositório no GIT

-- PARA INICIAR O PROJETO --

Apontar o diretório do projeto dentro do prompt
ex: 
cd c:\angular\projetos\nabag-app

Baixar o pacote npm com o seguinte comando:

npm install

Para as funcionalidades do projeto baixe os seguintes pacotes via npm:

-- SIMULADOR DE SERVIDOR VIA JSON --
npm install -g json-server

-- INSTALAÇÃO DOS PACOTES DO CARROSSEL --
npm install ngx-owl-carousel --save
npm install owl.carousel --save


-- FIREBASE --
npm install --save firebase
npm install @angular/fire --save

-- ANIMAÇÕES
npm install --save web-animations-js
npm install --save @angular/animations

-- INSTALAÇÃO DA NOTIFICAÇÃO DE ADICIONADO AO CARRINHO --
npm install ngx-toastr --save

-- INSTALAÇÃO NG-BOOTSTRAP --
npm install --save @ng-bootstrap/ng-bootstrap

Após instalar todos os pacotes 

-- PARA INICIAR A APLICAÇÃO --

ng serve

-- PARA SIMULAR O SERVIDOR/BD VIA JSON --

(executar em um prompt separado)
json-server --watch banco-de-dados.json

-- PARA VISUALIZAR NO NAVEGADOR --

(colocar no campo da url)
localhost:4200
